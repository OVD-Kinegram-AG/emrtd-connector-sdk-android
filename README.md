# Kinegram eMRTD Connector SDK Android

The Kinegram eMRTD Connector enables your Android app to read and verify
electronic passports / id cards ([eMRTDs][emrtd]). It enables the
[Document Validation Server (DocVal)][docval] to communicate with the
eMRTD through a secure WebSocket connection.

## Documentation

The documentation lives at [docs.kinegram.digital][docs]:

- [Overview][docs]
